# Index de documentation — Odoo Re-Watt ERP

Cette documentation décrit la maquette **Odoo Re-Watt ERP v0.5.1-docs**, basée sur la version fonctionnelle **v0.5.0 Repair & Knowledge métier**.

## Lecture recommandée

1. `INSTALLATION.md` — installer et lancer la maquette.
2. `UTILISATION_DEMO.md` — dérouler la démonstration direction.
3. `PROCESSUS_METIER.md` — comprendre les parcours métier couverts.
4. `MODULES.md` — comprendre les modules Odoo et custom.
5. `ARCHITECTURE.md` — comprendre la structure technique.
6. `ROADMAP.md` — voir les prochaines versions.
