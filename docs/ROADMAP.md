# Roadmap SAV Magasin Odoo

## Version courante : v0.2.0

Cette version ajoute le premier incrément métier exploitable autour du comptoir SAV :

- workflow SAV renforcé ;
- boutons d'action par statut ;
- assistant guidé de création de dossier SAV ;
- contrôles obligatoires au dépôt ;
- vue formulaire SAV plus orientée métier ;
- menu direct « Nouveau dossier » ;
- préparation du bon de dépôt PDF.

## Prochaines priorités

1. Relier les lignes pièces aux vraies réservations de stock.
2. Ajouter l'action « commander fournisseur depuis dossier SAV ».
3. Ajouter la réception d'une pièce et la notification du dossier en attente.
4. Créer une vue stock distant ergonomique par article.
5. Améliorer les documents PDF : dépôt, restitution, diagnostic.
6. Ajouter les règles de sécurité par rôle et par magasin.
