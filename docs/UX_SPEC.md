# Spécification UX — MVP

## Écran 1 — Nouveau dossier SAV

Champs visibles :

- Client
- Produit déposé
- Marque
- Modèle
- Numéro de série
- Garantie
- Symptôme client
- Accessoires déposés
- État visuel
- Magasin
- Responsable

Actions :

- Enregistrer brouillon
- Valider dépôt
- Imprimer bon de dépôt

## Écran 2 — Dossier SAV

Onglets :

- Résumé dépôt
- Diagnostic
- Pièces / stock
- Commercial
- Historique

Actions selon statut :

- Lancer diagnostic
- Attente client
- Attente pièce
- Réparer
- Prêt à restituer
- Marquer facturé
- Restituer
- Clôturer

## Écran 3 — Ligne pièce

Informations critiques :

- Article
- Quantité nécessaire
- Stock local
- Stock total
- Décision stock
- Réservé pour SAV

## Écran 4 — Réception sans commande

Étapes :

1. Fournisseur, BL, facture, magasin
2. Lignes articles
3. Contrôle
4. Validation métier

La V1 pose le modèle métier. La création réelle des pickings entrants sera à implémenter dans la phase suivante.
